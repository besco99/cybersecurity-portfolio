# VLAN Segmentation Lab

## Objective
Create multiple VLANs and isolate traffic between departments.

## Tasks Performed
- Created VLANs
- Assigned access ports
- Configured trunk ports
- Verified VLAN membership

## Key Commands
- vlan
- switchport mode access
- switchport trunk encapsulation dot1q
- show vlan brief

## Skills Demonstrated
- Network segmentation
- Broadcast domain isolation
- Switch configuration
