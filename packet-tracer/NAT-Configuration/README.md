# NAT Configuration Lab

## Objective
Implement Network Address Translation to allow internal hosts to access external networks.

## Tasks Performed
- Configured inside and outside interfaces
- Implemented static and dynamic NAT
- Verified translation table

## Key Commands
- ip nat inside
- ip nat outside
- ip nat inside source
- show ip nat translations

## Skills Demonstrated
- Private-to-public IP translation
- WAN simulation
- Internet access configuration
