# Deploy Devices & Create a Simple Network

## Objective
Deploy routers, switches, and end devices in Packet Tracer and establish basic LAN connectivity.

## Tasks Performed
- Added router, switch, and PCs
- Selected proper cable types
- Assigned static IP addresses
- Configured default gateway
- Verified connectivity using ping

## Verification Commands
- ping
- ipconfig
- show ip interface brief

## Skills Demonstrated
- Basic LAN topology design
- IP configuration
- Physical vs logical topology understanding
