# Inter-VLAN Routing Lab

## Objective
Enable communication between VLANs using Router-on-a-Stick configuration.

## Tasks Performed
- Created router subinterfaces
- Enabled 802.1Q encapsulation
- Assigned gateway IPs
- Verified routing between VLANs

## Key Commands
- interface g0/0.x
- encapsulation dot1Q
- ip address
- show ip route

## Skills Demonstrated
- Layer 3 routing
- Gateway configuration
- Packet forwarding validation
